IMAGES DIRECTORY STRUCTURE

This directory contains all images for the Agricultural and Environmental Engineering website.

Main Images:
- department-logo.png: Department logo displayed in navigation (supports .png, .jpg, .jpeg, .svg, .gif)
- hero-background.jpg: Main background image for homepage (department building recommended)
- about-department.jpg: Department overview image in About section
- ui_logo.svg: University of Ibadan logo (backup/reference)

Subdirectories:
- staff/: Staff and faculty photos
- gallery/: Department activity and achievement photos
- news/: News article images
- resources/: Resource-related images

Image Format Support:
All image tags now support multiple formats:
- .jpg / .jpeg (recommended for photos)
- .png (recommended for logos with transparency)
- .webp (modern format, smaller file sizes)
- .svg (for department logo, scalable)
- .gif (for simple graphics)

For the hero background image (hero-background):
- Use a high-quality photo of your departmental building
- Recommended size: At least 1920x1080 pixels
- Supported formats: .jpg, .jpeg, .png, .webp
- File size: Keep under 3MB for fast loading

For the department logo (department-logo):
- Use your official department logo
- Recommended formats: .png (with transparency) or .svg (scalable)
- Recommended size: At least 200x200 pixels
- File size: Keep under 500KB

To add/replace images:
1. Upload your image files to the appropriate subdirectory
2. Name them exactly as specified in the HTML comments
3. Ensure proper image formats (JPG, PNG)
4. Test the website to verify images display correctly
